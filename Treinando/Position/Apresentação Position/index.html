Desafio 2:
Criem uma estrutura HTML básica com um parágrafo

usem CSS para criar um layout específico com base nas seguintes diretrizes:

Selecione o parágrafo e defina-o como position: relative;

Dentro desse parágrafo, coloque um elemento filho (por exemplo, uma imagem) e defina-o como position: absolute;

Ajuste as propriedades auxiliares: top, right, bottom e left para posicionar o elemento filho dentro do elemento pai de acordo com um layout desejado

Observe como o elemento posicionado afeta o fluxo do documento e como ele é removido do fluxo normal Peça aos alunos que documentem suas observações e descubram como o uso da propriedade position afeta o

layout e a sobreposição de elementos
