Grid Layout

Grid Layout é uma função muito útil do HTML e CSS, ajuda com a separação de informações para facilitar a visualização das informações ao usuário.

Esse módulo, oferece linhas e colunas, facilitando a criação de páginas sem usar, "Float" e "Positioning" por exemplo.
