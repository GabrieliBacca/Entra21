Grid Elements

Grid Layout deve ter um elemento pai (container) com a propriedade display definida para grid ou inline-grid.

Os elementos filhos do grid container automaticamente serão grid items.

Grid Layout usando display: grid, exemplo:

batata
picles
maçã
banana
laranja
limão
hortelã
tomate
cebola

Grid Layout usando display: inline-grid, exemplo:

batata
picles
maçã
banana
laranja
limão
hortelã
tomate
cebola
